Helper function for executing a chain of Scrim script files with recursion tracking.

This function processes a list of file paths associated with a Chain command. It executes the commands within each script file in order, avoiding "open", "save" and "blank", recursively executing nested chain commands while avoiding infinite recursion.

To detect and avoid infinite recursion, a call stack (visited) is maintained, tracking the current chain of files being processed.
