Image * prog::command::RotateLeft::apply ( Image *  img)

Rotates the image 90 degrees counter-clockwise.

Creates a new image where the original's columns become rows and rows become columns. Then it assigns each (x, y) pixel to (y, nr. of columns - x), performing the left rotation.
