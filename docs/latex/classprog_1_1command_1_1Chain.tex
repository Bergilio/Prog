Helper function for executing a chain of \mbox{\hyperlink{classprog_1_1Scrim}{Scrim}} script files with recursion tracking. 

This function processes a list of file paths associated with a \mbox{\hyperlink{classprog_1_1command_1_1Chain}{Chain}} command. It executes the commands within each script file in order, avoiding \char`\"{}open\char`\"{}, \char`\"{}save\char`\"{} and \char`\"{}blank\char`\"{}, recursively executing nested chain commands while avoiding infinite recursion.

To detect and avoid infinite recursion, a call stack ({\ttfamily visited}) is maintained, tracking the current chain of files being processed. \mbox{\Hypertarget{classprog_1_1command_1_1Chain_ad5c7fac99b96ae311d1448b3a86bc295}\label{classprog_1_1command_1_1Chain_ad5c7fac99b96ae311d1448b3a86bc295}}
